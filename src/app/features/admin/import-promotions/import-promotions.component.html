Importar promoções

Cole um JSON com várias promoções. Use imageUrl público direto. Use publishAt para definir a data/hora exibida de publicação, útil para importações retroativas. Se omitido, a API usa a hora do processamento. Isso não agenda publicação futura — a promoção é importada já publicada. Formato ISO 8601 com fuso: 2026-06-27T21:30:00-03:00. Valide localmente ou faça Dry run antes de importar.
